_Commands_
CleverCord Currently supports the following commands

```
| General                                                                                       |                                    |
| --------------------------------------------------------------------------------------------- | ---------------------------------- |
| 8ball:                                                                                        | act as a Magic 8 Ball.             |
| coinflip:                                                                                     | flip a coin for you.               |
| invite: Sends a link to add the bot.                                                          |                                    |
| roll:                                                                                         | roll a dice for you.               |
| Extra                                                                                         |                                    |
| calculator: a calculator!                                                                     |                                    |
| gifembed: embeds a gif.                                                                       |                                    |
| link-shorten: Shortens a Link.                                                                |                                    |
| split: Sends split messages with a specific total length.                                     |                                    |
|                                                                                               |                                    |
| Fun                                                                                           |                                    |
| joke:                                                                                         | random one-liner joke.             |
| music: BROKEN:                                                                                | Plays requested music              |
| throw: Makes good use of the lemons                                                           |                                    |
|                                                                                               |                                    |
| Games                                                                                         |                                    |
| blackjack: Play a game of blackjack.                                                          |                                    |
| fishy: Go fishing.                                                                            |                                    |
| rock-paper-scissors: Play Rock-Paper-Scissors.                                                |                                    |
| roulette: Play a game of roulette.                                                            |                                    |
| slots: Play a game of slots.                                                                  |                                    |
| tic-tac-toe: Play a game of tic-tac-toe with another user.                                    |                                    |
|                                                                                               |                                    |
| Info                                                                                          |                                    |
| request: Send in a suggestion/request!                                                        |                                    |
| server:                                                                                       | send information about the server. |
| user-info: Gets information about a user.                                                     |                                    |
|                                                                                               |                                    |
| Moderation                                                                                    |                                    |
| ban: Bans a specified user.                                                                   |                                    |
| clear: Clears messages from the current channel.                                              |                                    |
| create-channel: Creates a channel.                                                            |                                    |
| kick: Kicks a specified user.                                                                 |                                    |
| mute: Disables a user's ability to talk.                                                      |                                    |
| prune: Prunes messages from a specified user.                                                 |                                    |
| softban: Softbans a specified user.                                                           |                                    |
| unban: Unbans a specified user.                                                               |                                    |
| unmute: Re-enables a user's ability to talk.                                                  |                                    |
|                                                                                               |                                    |
| Admin                                                                                         |                                    |
| poll:                                                                                         | start a poll for you.              |
| say:                                                                                          | repeat after you.                  |
|                                                                                               |                                    |
| Commands                                                                                      |                                    |
| groups: Lists all command groups.                                                             |                                    |
| enable: Enables a command or command group.                                                   |                                    |
| disable: Disables a command or command group.                                                 |                                    |
| reload: Reloads a command or command group.                                                   |                                    |
| load: Loads a new command.                                                                    |                                    |
| unload: Unloads a command.                                                                    |                                    |
|                                                                                               |                                    |
| Utility                                                                                       |                                    |
| help: Displays a list of available commands, or detailed information for a specified command. |                                    |
| prefix: Shows or sets the command prefix.                                                     |                                    |
| ping: Checks the bot's ping to the Discord server.                                            |                                    |
```
