_Commands_
MisCord Currently supports the following commands

```
|   Server Management Commands                                                                  |  Notes                             |
| --------------------------------------------------------------------------------------------- | ---------------------------------- |
| server-list: List all Managed Servers for the current Guild                                   |                                    |
| server-add: Add a server to this guild                                                        | m> help server-add to check params |
| server-delete: Remove a Specified server from this Guild                                      | requires serverId from server-list |
| server-info: Fetch server Status and player list for a specified server in this Guild         | requires serverId from server-list |
|                                                                                               |                                    |
|  Whitelist Management                                                                         |                                    |
| --------------------------------------------------------------------------------------------- | ---------------------------------- |
| whitelist-show                                                                                | requires serverId from server-list |
|                                                                                               |                                    |
|                                                                                               |                                    |
|   Owner|Developer Commands                                                                    |                                    |
| --------------------------------------------------------------------------------------------- | ---------------------------------- |
| groups: Lists all command groups.                                                             |                                    |
| enable: Enables a command or command group.                                                   |                                    |
| disable: Disables a command or command group.                                                 |                                    |
| reload: Reloads a command or command group.                                                   |                                    |
| load: Loads a new command.                                                                    |                                    |
| unload: Unloads a command.                                                                    |                                    |
|                                                                                               |                                    |
|   Utility Commands                                                                            |                                    |
| --------------------------------------------------------------------------------------------- | ---------------------------------- |
| help: Displays a list of available commands, or detailed information for a specified command. |                                    |
|                                                                                               |                                    |
| --------------------------------------------------------------------------------------------- | ---------------------------------- |
```
